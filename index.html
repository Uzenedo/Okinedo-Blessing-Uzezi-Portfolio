Below are the tools that i use for my Analysis

Microsoft Excel

It is a Software Program created by Microsoft that uses spreadsheets to organise Numbers and Data in columns and rows that can be Manipulated with Formulars and Functions. It contains Powerful Data Visualization and Analytical tools like Graphing tools and Pivot tables for Dashboarding.

SQL

Structured Query Language abbrevated as SQL is a domain specific language used in programming and designed for managing data held in a Relational Database Management System . It is a Structured Language for storing, manipulating and retrieving data in databases.

Power Bi

It is an interactive data Visualization software developed by Microsoft with a primary focus on Business Intelligence and Visualization tool that enables Data Analytics. It allows Professionals to process, analyse, extract insights, draw conclusions and share results in form of reports and dashboards.

Python

Python is a high-level Programming Language i.e it can be used to create variety of different programs and is not specialised for any specific problems. It is used in Machine Learning, Web Developments, Desktop Applications, Data Analysis and many other fields.
